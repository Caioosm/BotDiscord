Funcionamento do bot:

1 - Boas vindas inicial:  Quando um novo membro entra no servidor do Discord, ele é direcionado para um canal de boas-vindas específico.
Neste canal, o membro é informado sobre a necessidade de se registrar antes de acessar outros canais.

2 - Formulário de registro: O servidor pode usar um bot ou uma ferramenta de registro personalizada para criar um formulário que os novos membros precisam
preencher.O formulário pode incluir informações como nome de usuário, idade, interesses, entre outros, dependendo das necessidades do servidor.

3 - Aceitação de regras: Além do formulário, os novos membros geralmente são solicitados a ler e aceitar as regras e diretrizes do servidor.
sso é importante para garantir que todos os membros estejam cientes das expectativas de comportamento no servidor.

4 - Verificação: Após preencher o formulário e aceitar as regras, o novo membro é verificado pelo sistema. Isso geralmente é feito automaticamente,
mas em alguns casos, um moderador pode revisar manualmente as inscrições.

5 - Acesso Total: Uma vez verificado, o novo membro ganha acesso total ao servidor, incluindo os canais específicos de discussão,compartilhamento de conteúdo
e outras funcionalidades disponíveis.
